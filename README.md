# Praticando ETL Capacidade de pagamento
## Resumo:
Projeto desenvolvido em python para prática de ETL. Sistema extrai dados de um arquivo csv. Transforma dados para produção de nova informação. Carrega novos dados adicionando ao arquivo csv.
### Tecnologias utilizadas:
- Python
- Ambiente (google colab)
- Biblioteca Pandas
